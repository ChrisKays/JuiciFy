JuiciFy
============

JuiciFy is a small eCommerce website where users can purchase different juice online.

Features
--------
-	Register : Screen to register a user.
-	Login : Screen to login
-	Home Screen: The home screen displays all the product and has the ability to filter the product.
-	Content management: we have 2 options under content management. PLEASE NOTE : ONLY LOGGED IN USER CAN VIEW THIS OPTION.
	1. Category: The category of the product
	2. Product: The actual product to sale.
-	Details: Products are grouped per category. User can click on the product to place their order.

Tools
--------
-	Visual Studio
-	ASP .NET Core Entity Framework
-	Razor Page
-	Bootstrap (SweetAlert)
- 	CSS
-	JavaScript
-	SQL Server 2019
